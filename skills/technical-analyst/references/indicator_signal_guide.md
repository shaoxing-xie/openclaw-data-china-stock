# Indicator signal guide

This note is used by the skill prompt as interpretation reference.

## Core mappings

- RSI > overbought: potential overheating, watch for reversal evidence.
- RSI < oversold: potential exhaustion, watch for trend-confirmation evidence.
- MACD DIF crossing DEA upward: momentum improvement signal.
- ADX > strong_trend: trend regime is stronger; prefer trend-following interpretation.
- Price near Bollinger upper band: strong momentum or short-term stretch; require counterevidence.
