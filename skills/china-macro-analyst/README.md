# china-macro-analyst

Macro analysis skill for China macro datasets.

## Quick usage

1. Call `tool_fetch_macro_snapshot` for first-pass context.
2. Call `tool_fetch_macro_data` for missing core datasets.
3. Produce four-section narration:
   - macro state
   - marginal changes
   - asset implication (bias only)
   - risk and counter-evidence

## Configuration

- `config/macro_config.yaml`: thresholds, quadrant rules, and mapping.
